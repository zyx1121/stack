---
title: Google OAuth
description: Apply Google OAuth to your project
---

1. Go to [Google Cloud Console | Create Project](https://console.cloud.google.com/projectcreate), and create a new project.

   <img src="images/google-oauth/01.webp" alt="step 1" />

2. Search for "Oauth console screen" in the search bar, and click on the first result.

   <img src="images/google-oauth/02.webp" alt="step 2" />

3. Click on the "GET STARTED" button.

   <img src="images/google-oauth/03.webp" alt="step 3" />

4. Enter the `App Information` section.

   <img src="images/google-oauth/04.webp" alt="step 4" />

5. Select the `External` option.

   <img src="images/google-oauth/05.webp" alt="step 5" />

6. Input the `Contact Information` section.

   <img src="images/google-oauth/06.webp" alt="step 6" />

7. Check the `Google API Services User Data Policy` and click on the `CREATE` button.

   <img src="images/google-oauth/07.webp" alt="step 7" />

8. Now search for "Credentials" in the search bar, and click on the first result.

   <img src="images/google-oauth/08.webp" alt="step 8" />

9. Click on the `Create Credentials` button, and select the `OAuth client ID` option.

   <img src="images/google-oauth/09.webp" alt="step 9" />

10. Follow the Screen Shot below to create the OAuth client ID.

    <img src="images/google-oauth/10.webp" alt="step 10" />

11. Now you can see the OAuth client ID and client secret! Please save them in a safe place.

    <img src="images/google-oauth/11.webp" alt="step 11" />
